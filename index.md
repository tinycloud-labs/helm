# TinyCloud Labs Helm Charts

Add this Helm repo:

```bash
helm repo add tinycloud-labs https://tinycloud-labs.github.io/helm
helm repo update
```

## Available Charts
| [gha-runner-deployment](https://github.com/tinycloud-labs/helm/tree/main/charts/gha-runner-deployment) | 0.0.1 | A Helm chart for Installing Github Actions runner deployment |
| [nfspersistence](https://github.com/tinycloud-labs/helm/tree/main/charts/nfspersistence) | 0.3.1 | A generic Helm chart for deploying NFS based persistence storage. |
| [unichart](https://github.com/tinycloud-labs/helm/tree/main/charts/unichart) | 0.0.5 | A generic Helm chart, supports stateful and statless applications. |
